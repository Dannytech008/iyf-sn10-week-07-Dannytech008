## Project Description
This project is a simple and interactive To-Do List web app built to practice DOM manipulation and event handling in JavaScript. Users can add tasks, mark tasks as completed, delete tasks, and view live task statistics.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- Browser localStorage

## Features
- Add new tasks through a form input
- Mark tasks as completed by clicking on task text
- Delete tasks from the list
- Live counters for Active, Completed, and Total tasks
- Persistent task storage using localStorage

## How to Run
1. Clone this repository
2. Open `to-do-list.html` in your browser

## Lessons Learned
- How to select and manipulate DOM elements dynamically
- How to handle user interactions with form, click, and keyboard events
- How to structure JavaScript code into reusable functions
- How to persist app data using localStorage

## Challenges Faced
- Some text was not visible due to color contrast and style overrides from JavaScript
- Keeping task state (active/completed) synchronized with the DOM and localStorage
- Updating task counters in real time after add, toggle, and delete actions
